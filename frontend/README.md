# 高考志愿填报辅助工具

一个基于 Vite + React + TypeScript 的前端应用，帮助高考考生筛选符合条件的大学，并查看往年在福建的录取情况。

## 功能特性

### 大学筛选
- **地区筛选**：按省份、城市筛选
- **城市等级筛选**：一线、新一线、二线、三线、四线、五线
- **层次筛选**：本科、专科（默认本科）
- **性质筛选**：公办、民办（默认公办）
- **搜索功能**：按大学名称、城市、省份搜索

### 大学详情
- 基本信息（省份、城市、城市等级、层次、性质）
- 宿舍条件（上床下桌、几人间、空调、独立卫浴等）
- 管理情况（断电断网、查寝、门禁等）
- 网络设施（校园网速度、价格）
- 交通出行（地铁、市区距离、交通便利性）
- 生活服务（外卖、食堂、超市、快递）

### 福建录取数据
- 2023-2025 福建分数线（历史组/物理组）
- 2026 招生计划（历史组/物理组）

## 技术栈

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite
- **样式**：TailwindCSS
- **数据**：JSON（从 Excel 转换）

## 快速开始

### 安装依赖
```bash
cd frontend
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:5173 查看应用。

### 构建生产版本
```bash
npm run build
```

## 数据说明

### 大学数据
- 来源：`Colleges/全国大学宿舍情况及管理情况.xlsx`
- 包含 3057 所大学的详细信息
- 35 个字段，涵盖宿舍条件、管理设施、生活服务等

### 福建录取数据
- 来源：`FuJian/` 目录下的 PDF 文件
- 2023-2025 福建分数线（历史组/物理组）
- 2026 招生计划（历史组/物理组）

## 使用说明

1. 启动应用后，默认显示所有公办本科大学
2. 使用左侧面板筛选条件
3. 点击大学查看详情
4. 在详情页面查看福建录取数据

## 注意事项

- 福建录取数据为 PDF 文件，需要手动打开查看
- 宿舍条件和管理设施数据格式不一致，在详情页展示而非作为筛选条件
- 城市等级筛选可独立于省份城市使用
